All bot commands:

-play or -p : Use this command and the name of the song for play it.

-pause : Use this command for stop the music.

-resume : Use this command for resuming the music.

-loop or -l: Use this command for toggle music loop.

-np : Show now playing song.

-queue or -q : Show the music queue and now playing.

-skip or -s : Use this command for skip the song.

-skipto or -st : Use this command for skip to the selected queue number.

-playlist or -pl : Use this command for play a playlist from YouTube.

-ping : Show the bot average ping.

-uptime or -u : Check the uptime.

-shuffle: Shuffle the queue.

-volume or -v : Change volume of currently playing music.

-invite : Send bot invite link.

-move or -mv : Move songs around in the queue.

-search : Search and select videos to play.
